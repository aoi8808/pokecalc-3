# ポケモン ダメージ計算ツール

## 概要
ポケモンの対戦におけるダメージ計算を、ブラウザ上で直感的に行えるWebアプリケーションです。CSVデータを利用して数百匹のポケモンや技のデータを管理し、素早く検索・計算結果を表示できるように設計しました。

## 使用技術
* HTML / CSS
* JavaScript (Vanilla JS)
* データ管理: CSVファイル
* 画像取得: PokeAPI、及び外部サイトからの引用

## こだわったポイント・工夫した点

* **データの動的処理:** 数百行に及ぶポケモンのステータスや技のCSVデータをJavaScriptでパースし、ドロップダウンリストに動的に反映させる仕組みを構築しました。
* **ユーザー体験（UX）の向上:** テキスト入力による絞り込み機能や、HPの減少割合を視覚的に表示するカラーバーを実装し、直感的に操作できるようにしました。
* **例外へのエラーハンドリング:** 画像データが存在しない新ポケモンなどが選択された際、プログラムが停止しないよう `onerror` を用いて代替画像や非表示処理を行うフォールバックを実装しました。

## ローカルでの動かし方
1. 本リポジトリをクローン、またはダウンロードします。
2. VS Codeの拡張機能「Live Server」等を使用して `index.html` を開きます。
（※ローカルサーバーを立ち上げないと、CSVファイルの読み込み（fetch）でCORSエラーが発生するため）
